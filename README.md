## Installation & Use

The [Wiki](https://github.com/TIBCOSoftware/spotfire-mod-basictextarea/wiki) contains step-by-step instruction on how to install and use this Mod in TIBCO Spotfire速.

## Building the Project

In a terminal window:
- `npm install`
- `npm run build-watch`

In a new terminal window:
- `npm run server`

### Building for production

The development version of bundle.js is uncompressed and not suitable for end-users. Run the following command to compress the bundle:
- `npm run build`
